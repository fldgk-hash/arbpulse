## How BSC Arbitrage Works

1. **Discovery** вҖ” DexScreener boosts + profiles + search filtered for `chainId: bsc`
2. **Batch fetch** вҖ” Up to 45 tokens, 30 per request across PancakeSwap, Uniswap BSC, THENA, Biswap, etc.
3. **Grouping** вҖ” Tokens trading on 2+ DEXs are grouped for cross-DEX comparison
4. **Net spread calc** вҖ” Real fees per DEX + 0.5% slippage applied
5. **Filtering** вҖ” Min liq / vol / spread filters applied; low-TVL pairs flagged
6. **Address display** вҖ” BSC pair addresses shown with BSCscan links for on-chain verification

## вҡ пёҸ Risk Disclaimer (DYOR)

> **DO YOUR OWN RESEARCH.**  
> ArbPulse Pro is an informational tool. It does **not** execute trades.  
>
> Arbitrage involves real-world risk:
> - **Slippage** вҖ” especially acute on low-TVL pairs (< $10K flagged in yellow)
> - **MEV bots** вҖ” sandwich attacks are common on both Solana and BSC

> - **BSC gas costs** вҖ” factor BNB gas into net profit calculations
> - **CEX withdrawal delays** вҖ” cross-exchange arb requires capital pre-positioned
